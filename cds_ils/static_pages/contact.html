Catalogue and loans questions

Please contact the CERN Library by email at library.desk@cern.ch for any question related to:

  • catalogue: how to find documents, different versions and editions or to get more details
  • e-books: how to get access or download
  • loaning: how to request, extend, return or cancel a loan
  • purchase or InterLibrary loans: how to buy a book or obtain it from another library
  • issues with the EDH termination sheets

You can also reach us directly at the Library desk, Monday-Friday from 9 a.m. to 6 p.m. in CERN building 52 or via telephone +41 22 767 2444.

Technical Support

For any question related to the technical issues with the website (error, bug, feature not working, etc...), please contact CDS support. In your message, please provide:

  • the URL where the issue has been found, with the error message and code if available
  • a description of the issue with any relevant information on how to reproduce it
  • if possible and useful, a screenshot
